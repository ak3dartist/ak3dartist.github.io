About the project

Stylized Hammer

A Stylized Hammer game assset inspired from Thor's Hammer. First I created the lowpoly model and sculpted the details on a highpoly mesh in blender. Then I UV Mapped the model and exported it into Substance Painter for baking the mesh maps. After baking I imported them into blender for Texturing. Then I added some lights in the scene for rendering.

Software used - Blender, Substance Painter
