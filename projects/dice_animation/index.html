About the project

Dice Animation

A simple 3D animation that I created within an hour. This scene contains two dice rolling in slow motion. I created the Models using simple subdivison modelling and textured them using the Shader Editor. For animating the dice I used Rigidbody physics system in Blender for simulating the dice roll. Then I quickly added some lights and rendered the animation in a sequence using Blender's real-time rendering engine Eevee.

Software used - Blender
